Sistem Quest Manager dan Player Manager ini dibuat untuk memenuhi tugas besar praktikum struktur data

Menu yang tersedia:

1. Create Main Quest
2. Create Subquest
3. Update Quest
4. Delete Quest
5. Show All Quests
6. Register Player
7. Player Login
8. Assign Quest to Player
9. Remove Player Quest
10. Show All Players
0. Exit

untuk menjalankan , buka terminal dan ketik `./main.exe`
